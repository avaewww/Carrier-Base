# Pixel Network Status 🌐

A centralized dashboard tracking the real-world support of modern network technologies (5G, VoLTE, VoWiFi) on Google Pixel devices across global mobile carriers. 

## 📌 The Problem
Missing 5G, VoLTE, or VoWiFi on your Pixel in an "unsupported" country? Often, the core issue lies with local mobile carriers who fail to provide the necessary IMS profiles or refuse to let these devices on their advanced networks. This project maps out which operators worldwide actually support Pixel devices.

## ✨ Features
* **Interactive 3D Globe:** Visually explore network support across different countries.
* **Dynamic Carrier Database:** Quickly check if your local provider supports essential network features.
* **Minimalist Dark UI:** A sleek, high-contrast design focused on readability.

## 📂 Data Management & Updates
To ensure maximum accuracy and prevent database corruption or spam, **direct database modifications by users are not permitted**. 

The database is strictly maintained and updated exclusively by the repository owner. All data is managed centrally, ensuring that only verified information goes live on the website. 

## 📢 How to Report a Carrier Status
I rely on the community to gather accurate data! If you have confirmed network status information for your local carrier (especially if you recently managed to get VoLTE/5G working on your device), please share it.

To submit an update:
1. Gather proof (e.g., screenshots of your status bar or network settings).
2. Send the information directly to my Telegram: **[@avaewww](https://t.me/avaewww)**.
3. I will review the proof and manually update the global database.
